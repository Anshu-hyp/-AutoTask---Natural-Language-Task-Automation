# -AutoTask---Natural-Language-Task-Automation
Convert plain English commands into automated workflows across your computer.


Tech Stack: Python, LangChain, Selenium, PyAutoGUI, OpenAI API
Features:

Natural language input: "Every Monday, download my bank statement and save to Dropbox"
Converts text to executable automation scripts
Browser automation for web tasks
Desktop automation for local applications
Schedule manager with cron-like functionality
Visual workflow builder as alternative to text
Learning mode: watches you perform tasks and creates automations

Unique Aspects: Zero-code automation through conversational AI with self-learning capability
